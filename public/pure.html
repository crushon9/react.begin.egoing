deploy

원본을 그대로 배포하면 파일용량이 크고 보안문제가 있음

  • ​npm run build 명령 : build 폴더 안에 용량이 줄어든 코드파일 자동생성
  • npm install -g serve : 서브설치 (npm을 통해 설치가능한 간단한 웹서버)
  • serve -s build : 서브를 실행시킬때 build 디렉토리를 루트로 하겠다
